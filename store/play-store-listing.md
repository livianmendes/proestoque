# ProEstoque

## Titulo

ProEstoque

## Descricao curta

Controle produtos, categorias e alertas de estoque em um app simples e direto.

## Descricao completa

ProEstoque e um aplicativo para organizacao de estoque. Ele permite cadastrar produtos, associar categorias, acompanhar quantidades, visualizar alertas de estoque baixo e manter os dados sincronizados com a API do projeto.

Principais recursos:

- Login e cadastro de usuarios
- Dashboard com resumo do estoque
- Lista de produtos com busca e filtros
- Cadastro, edicao e exclusao de produtos
- Alertas de estoque baixo
- Integracao com API propria

## Palavras-chave

estoque, produtos, inventario, controle, categorias, reposicao
