# Politica de Privacidade - ProEstoque

Ultima atualizacao: 25 de junho de 2026.

O ProEstoque utiliza dados fornecidos pelo usuario para autenticar o acesso e permitir o gerenciamento de produtos no aplicativo.

## Dados coletados

- Nome
- E-mail
- Senha criptografada no servidor
- Produtos cadastrados pelo usuario durante o uso do app

## Uso dos dados

Os dados sao usados para login, cadastro, recuperacao de sessao e exibicao das informacoes de estoque dentro do aplicativo.

## Compartilhamento

O ProEstoque nao vende nem compartilha dados pessoais com terceiros para fins comerciais.

## Armazenamento

Os dados sao armazenados na API do projeto. A senha e protegida por criptografia antes de ser salva no banco de dados.
