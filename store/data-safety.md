# Data Safety - ProEstoque

Resumo para preencher o formulario de seguranca de dados da Google Play.

## Dados coletados

- Nome: usado para identificacao do usuario no app.
- E-mail: usado para login e identificacao da conta.
- Senha: usada para autenticacao, armazenada criptografada no servidor.
- Dados de produtos: usados para exibicao e controle do estoque do usuario.

## Finalidade

- Autenticacao de conta
- Funcionalidade principal do aplicativo
- Gerenciamento de estoque

## Compartilhamento com terceiros

Nao ha compartilhamento de dados com terceiros para publicidade ou venda de informacoes.

## Criptografia em transito

Em producao, a API deve usar HTTPS para proteger os dados em transito.

## Exclusao de dados

O usuario pode solicitar a exclusao dos dados pelo e-mail de contato informado na politica de privacidade.
